### Trying Sample
Download the sample.

    $ git clone git://github.com/parroty/anglarjs_sample_rails.git
    $ cd anglarjs_sample_rails

Install prerequisite component.

    $ bundle

Migrate database

    $ rake db:migrate

Launch server.

    $ rails s

Open the url with the browser

    $ open http://localhost:3000
